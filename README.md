# Health Reminder App

A zero-cost mobile notifier app that reminds users to walk, stand, drink water, stretch, and take healthy breaks while working.

## Vision
Help desk workers build healthier habits through lightweight recurring reminders.

## Core Features
- Local notifications every X hours
- Reminder categories: walk, water, stand, stretch, eye break
- Settings for schedule and frequency
- History and streak tracking
- Offline-first and privacy friendly

## Tech Stack
- React Native
- Expo
- expo-notifications
- AsyncStorage

## Roadmap
See `/docs/ROADMAP.md`

## Getting Started
```bash
npm install
npx expo start
```

Scan QR code with Expo Go on Android.
